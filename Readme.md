# Document and Code Agent

An AI-powered agent designed to **generate both code and documentation** for software projects, helping developers move from idea to implementation faster.

## Live Demo
https://ai-code-doc-helper.netlify.app/

## Overview
This project allows users to input a coding task or concept, and the AI agent automatically creates:
- High-quality **source code**
- Clear and structured **documentation**

It aims to simplify development workflows by combining intelligent code generation with contextual explanations.

## Features
- Generates working code from prompts and documentation
- Automatically documents functions, classes, and modules  
- Can export results
